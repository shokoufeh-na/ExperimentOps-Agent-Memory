ExperimentOps Agent
AI Agent with Persistent Memory using CockroachDB × AWS

Overview
ExperimentOps Agent is an AI agent built for the CockroachDB × AWS Hackathon. It demonstrates how agentic applications can use CockroachDB as a durable, production‑grade memory layer and AWS for scalable execution. The agent stores experiment history, retrieves relevant past runs, analyzes failures, and recommends next steps — showing how persistent memory makes agents genuinely useful in real workflows.
